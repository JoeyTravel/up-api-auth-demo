UP-api-auth-demo

Demo of Jawbone UP authentication and API access using passport and the jawbone-up Javascript API wrapper

Download .zip Download .tar.gz View on GitHub

This is a demonstration of authenticating and using the Jawbone API using passport, node-jawbone-up and storing the user and access token in a mongo database using mongoose. It includes authenticating and pulling the name of the user and their steps and sleep.

For docs on using the nose-jawbone-up wrapper, passport, or mongoose please see the links below and/or their web sites.

To install the mongo db go to http://ww.mongodb.org/downloads
